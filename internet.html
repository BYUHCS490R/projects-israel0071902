internet

The internet is a very big system that connects lot of computers that are in all over the world. Because they are connected to each other , computers can send and receive lot of information faster and easier.

web

The web is a part of the internet. It is made of lot websites and web pages that we open using a web browser, like Chrome, Firefox, or Safari.

How are the Internet and Web different?

The internet is a network of computers that are connected to all computers together.

The web is a collection of websites that we can visit using the internet.

ASCII and Unicode

ASCII is a type of way to represent text using simple numbers. It can be included using letters, numbers, and some symbols.

Unicode is a bigger system that can helps and represent text from every language in the world.

What is UTF-8

UTF-8 is a way to store text using Unicode. It can handle all kinds of characters from different languages and it can be used lot often on the web.

URL

A URL is the address you type in the browser to find a specific website or page.

Domain Name

A domain name is the easy to remember the name of a website, like google.com.

HTTP

HTTP is the system that helps my browser to request information from a website and show it to my screen.

HTTPS

HTTPS is the secure and safer version of HTTP. It protects the information sent between my computer and the website.

IP Address

An IP address is a unique number that identifies a computer or device on the internet.

DNS

DNS is like a library for the internet. It changes website names into the numbers like IP addresses that computers understand.

W3C Recommendations

W3C Recommendations are official rules and guidelines for building websites, made by a group that manages how the web works.

XML, XHTML, HTML, HTML5

XML is a way to organize and share data in a structured format.

XHTML is a strict version of HTML that needs to follow certain rules.

HTML is the basic language used to create web pages.

HTML5 is the latest version of HTML with new features for building better, more interactive websites.
